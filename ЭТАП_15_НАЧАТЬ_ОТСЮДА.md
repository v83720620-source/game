## 🎯 ЧТО НУЖНО СДЕЛАТЬ В UNITY:

### ДЕНЬ 1: БАЗОВАЯ НАСТРОЙКА (30 минут)

#### ШАГ 1: Добавьте Managers в сцену

```
Unity → Scenes → Game

1. CREATE EMPTY GAMEOBJECTS:
   - NetworkMatchManager
   - NetworkTeamManager
   - NetworkSpawnManager
   - NetworkDuelMode (если играете Duel)
   - NetworkTDM3v3Mode (если играете TDM)

2. ADD SCRIPTS:
   - NetworkMatchManager → Network Match Manager (Script)
   - NetworkTeamManager → Network Team Manager (Script)
   - NetworkSpawnManager → Network Spawn Manager (Script)
   - NetworkDuelMode → Network Duel Mode (Script)
   - NetworkTDM3v3Mode → Network TDM3v3 Mode (Script)
```

---

#### ШАГ 2: Создайте Spawn Points

```
Hierarchy → Create Empty:

1. TEAM 1 SPAWNS:
   Create Empty → "Team1Spawns"
   Под ним создайте 4-6 пустых объектов:
   - Team1_Spawn_1: Position (-10, 1, -10)
   - Team1_Spawn_2: Position (-8, 1, -10)
   - Team1_Spawn_3: Position (-10, 1, -8)
   - Team1_Spawn_4: Position (-8, 1, -8)

2. TEAM 2 SPAWNS:
   Create Empty → "Team2Spawns"
   Под ним создайте 4-6 пустых объектов:
   - Team2_Spawn_1: Position (10, 1, 10)
   - Team2_Spawn_2: Position (8, 1, 10)
   - Team2_Spawn_3: Position (10, 1, 8)
   - Team2_Spawn_4: Position (8, 1, 8)

3. NEUTRAL SPAWNS (для Duel):
   Create Empty → "NeutralSpawns"
   Создайте 6-8 точек по карте:
   - Neutral_Spawn_1: Position (0, 1, 15)
   - Neutral_Spawn_2: Position (0, 1, -15)
   - Neutral_Spawn_3: Position (15, 1, 0)
   - Neutral_Spawn_4: Position (-15, 1, 0)
   - Neutral_Spawn_5: Position (10, 1, 10)
   - Neutral_Spawn_6: Position (-10, 1, -10)
```

---

#### ШАГ 3: Настройте NetworkSpawnManager

```
NetworkSpawnManager → Inspector:

Team 1 Spawn Points: (Size: 4-6)
  - Перетащите все дочерние объекты Team1Spawns

Team 2 Spawn Points: (Size: 4-6)
  - Перетащите все дочерние объекты Team2Spawns

Neutral Spawn Points: (Size: 6-8)
  - Перетащите все дочерние объекты NeutralSpawns

Min Spawn Distance: 10
Random Spawn: ✅
```

---

#### ШАГ 4: Настройте Game Modes

**Для Duel 1v1:**
```
NetworkDuelMode → Inspector:
  Kills To Win: 5
  Match Duration: 180
  Overtime Duration: 60
  Respawn Delay: 3
  Min Spawn Distance From Enemy: 20
```

**Для TDM 3v3:**
```
NetworkTDM3v3Mode → Inspector:
  Kills To Win: 40
  Match Duration: 420
  Max Players Per Team: 3
  Respawn Delay: 5
```

---

#### ШАГ 5: Сохраните сцену

```
Ctrl + S (Save Scene)
```

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### Ошибка: "NetworkMatchManager.Instance is null"
```
Решение:
1. Убедитесь что NetworkMatchManager GameObject есть в сцене
2. У него есть NetworkObject component
3. Сцена сохранена (Ctrl + S)
```

### Ошибка: "No spawn points found"
```
Решение:
1. Проверьте что Team1Spawns, Team2Spawns, NeutralSpawns существуют
2. Проверьте что у них есть дочерние объекты (spawn points)
3. Проверьте что spawn points назначены в NetworkSpawnManager Inspector
```

### Ошибка: "Players spawning at (0,0,0)"
```
Решение:
1. Spawn points не назначены в Inspector
2. Или AutoSetupSpawnPoints() не сработал
3. Проверьте Console логи
```
