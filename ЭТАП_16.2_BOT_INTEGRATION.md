## 🔧 ИНТЕГРАЦИЯ С MATCHMAKING:

Теперь когда QueueSystem готов к запуску матча, он должен spawнить ботов:

```csharp
// В MatchmakingUI.cs или NetworkMatchManager.cs:

private void OnMatchReady()
{
    // 1. Запускаем сервер
    CustomNetworkManager.Instance.StartHost();
    
    // 2. Ждём инициализацииyield return new WaitForSeconds(1f);
    
    // 3. Spawnim ботов
    int botsNeeded = _queueSystem.PlayersNeeded - _queueSystem.PlayersFound;
    
    if (botsNeeded > 0 && NetworkBotSpawner.Instance != null)
    {
        // Распределяем ботов по командам
        int botsPerTeam = botsNeeded / 2;
        
        NetworkBotSpawner.Instance.SpawnBots(botsPerTeam, teamId: 1);
        NetworkBotSpawner.Instance.SpawnBots(botsPerTeam, teamId: 2);
    }
    
    // 4. Загружаем сцену
    SceneLoader.Instance.LoadScene("Game");
}
```
