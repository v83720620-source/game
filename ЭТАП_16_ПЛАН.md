## 🛡️ ЭТАП 16.4: ANTI-FRUSTRATION AI (1 день)

### **Что нужно реализовать:**

#### **1. Fair Play Mechanics**
```
No Wallhacks:
├── Bot "forgets" player behind walls
├── Last Known Position tracking
├── Line of Sight check (raycast)
└── Memory timeout (3s)

Smoke Respect:
├── Bot doesn't shoot through smoke accurately
├── Random spread increase in smoke
└── No perfect tracking through smoke

360 No-Scope Ban:
├── Bot needs time to turn (realistic)
├── Aim smoothing (Lerp rotation)
└── Min turn time: 0.2-0.5s
```

#### **2. Mercy Rule**
```
Player protection:
├── Track player deaths vs bot
├── If player died 3+ times in a row → reduce bot difficulty
├── Temporary difficulty drop (-1 level)
├── Reset after player gets kill
└── Subtle (player doesn't notice)
```

#### **3. Skill Balancing**
```
Dynamic adjustment:
├── Track K/D ratio per bot
├── If bot too strong → reduce accuracy
├── If bot too weak → increase reaction time
└── Target: 0.8-1.2 K/D
```

## 🎮 СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### **Scenario 1: Quick Match (Solo Player)**
```
1. Player открывает Lobby
2. Выбирает "Duel 1v1"
3. Нажимает [Find Match]
4. Система ищет 15s
5. Timeout → добавляет 1 бота
6. Match starts: Player vs Bot
```

### **Scenario 2: Matchmaking с игроком**
```
1. Player A joins queue (TDM 3v3)
2. Player B joins queue (TDM 3v3)
3. Wait 15s for more players
4. Timeout → add 4 bots
5. Match starts: 2 players + 4 bots (3v3)
```

### **Scenario 3: Player Disconnect**
```
1. Match in progress (TDM 3v3)
2. Player disconnects (Alt+F4)
3. System waits 30s for reconnect
4. Timeout → spawn bot in their place
5. Bot continues playing for their team
```

### **Scenario 4: Mercy Rule**
```
1. Player dies 3 times in a row to bot
2. System detects frustration
3. Bot difficulty drops: Hard → Normal
4. Player gets kill
5. Difficulty resets back to Hard
```
