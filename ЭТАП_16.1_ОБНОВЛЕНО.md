## 🎯 КАК РАБОТАЕТ:

### **Сценарий 1: Practice Mode (Solo + Bots)**
```
1. User выбирает режим → Lobby Scene
2. Нажимает "Start" (автоматически для Practice)
3. QueueSystem.StartSearch(playersNeeded)
4. Searching... (0-15s)
5. Timeout 15s → BotFilling
6. Match Ready! → LoadScene("Game")
7. Боты спавнятся автоматически
```

### **Сценарий 2: Multiplayer (с ботами если нужно)**
```
1. User выбирает Duel/TDM → Lobby Scene
2. Ждёт других игроков (0-15s)
3. Если нашлись → Match starts
4. Если нет → Timeout 15s → добавляются боты
5. Match starts с игроками + ботами
```
