## 🔧 КАК РАБОТАЕТ:

### **Пример: Duel 1v1**

```
1. Player открывает LobbyScene
2. Нажимает [Duel 1v1]
3. MatchmakingManager.JoinQueue() → RPC to server
4. Server: QueueSystem.JoinQueue()
5. Server: Players: 1/2 → ищет игроков
6. Timeout 15s → добавляет 1 бота
7. Match ready! → LoadScene("GameScene")
8. NetworkGameModeManager спавнит игроков + ботов
```

### **Таймлайн:**
```
0s   - Player joins queue
      "Searching for players..."
      Players: 1/2

15s  - Bot fill timeout
      "Adding bots..."
      Players: 1/2 + 1 bot

16s  - Match starting!
      Scene loading...

20s  - In game!
```
