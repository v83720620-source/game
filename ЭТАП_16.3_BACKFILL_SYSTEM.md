## 🎯 КАК РАБОТАЕТ:

### **Scenario 1: Player Disconnect**
```
1. Player теряет соединение (Alt+F4, internet loss, etc.)
2. BackfillManager.OnClientDisconnected() triggered
3. Сохраняет данные игрока (team, kills, deaths, name)
4. Запускает 30s timer
5. Ждёт reconnect...

ЕСЛИ reconnect:
  → Таймер отменяется
  → ReconnectHandler восстанавливает состояние
  → Game continues!

ЕСЛИ НЕТ reconnect:
  → Timeout 30s
  → BackfillManager spawns бота на ту же команду
  → Бот занимает место игрока
  → Game continues seamlessly!
```

### **Scenario 2: AFK Player**
```
1. Player не двигается 45s
2. AFKDetector → Show warning: "You will be kicked!"
3. Если активность возобновилась → Timer reset
4. Если 60s без активности:
   → AFKDetector.RequestKickServerRpc()
   → Server disconnects player
   → BackfillManager → Spawn bot
   → Game continues!
```

### **Scenario 3: Reconnect**
```
1. Player disconnect (30s таймер запущен)
2. Player reconnects через 15s
3. ReconnectHandler.OnClientConnected()
4. Проверка: CanPlayerReconnect(clientId)
5. Восстановление:
   - Team assignment
   - Position
   - Kills/Deaths
   - Loadout
6. Notification: "Player X reconnected!"
7. Game continues!
```
