## 🎯 КРАТКОЕ ОПИСАНИЕ ПРОЕКТА

Разработал полноценный мобильный 5v5 team shooter "Flump Game" с использованием Unity 6.3 LTS и Universal Render Pipeline. Игра включает систему управления персонажем, AI ботов, два игровых режима (Team Deathmatch и Hardpoint), мобильный UI и оптимизацию для слабых устройств.

---

## 🛠️ ЧТО БЫЛО СДЕЛАНО

### **ЭТАП 1-2: ОСНОВА ИГРЫ**
- Настроил Unity 6.3 LTS + URP 17.3.0 + Input System 1.17.0
- Создал систему управления персонажем от первого лица (движение, бег, прыжки, приседание)
- Реализовал First Person Camera с плавным управлением

### **ЭТАП 3: UI И ВИЗУАЛ**

### **ЭТАП 4: БОЕВАЯ СИСТЕМА**

- Настроил зоны попадания (голова x2.0, тело x1.0, конечности x0.75)

### **ЭТАП 5-6: AI И КОМАНДЫ**

### **ЭТАП 7-8: ИГРОВЫЕ РЕЖИМЫ**

## 📊 ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

**Архитектура:**
- Singleton pattern для менеджеров (GameManager, AudioManager, VFXManager, QualityManager)
- ScriptableObject для данных оружия
- Event-driven система коммуникации
- Component-based дизайн

**Оптимизация:**
- Object pooling для VFX и Audio
- NavMesh для AI навигации
- Кэширование компонентов
- Асинхронные операции
- 3 уровня качества графики

**Платформы:**
- Android (API 24+, ARM64, IL2CPP)
- iOS ready (настройки готовы)
