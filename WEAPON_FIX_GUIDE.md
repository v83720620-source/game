## ✅ ИСПРАВЛЕНИЕ 1: РАЗМЕР И ПОЗИЦИЯ ОРУЖИЯ (2 минуты)

### Вариант A: Правильные настройки для FPS оружия

```
1. Unity → Project → Prefabs → Network → Player.prefab
2. Hierarchy → Player → Main Camera → Weapon → WeaponModel
3. Inspector → Transform:

Position:
  X: 0.4      (правее)
  Y: -0.3     (ниже)
  Z: 0.4      (ближе к камере)

Rotation:
  X: 0
  Y: 0
  Z: 0

Scale:
  X: 0.05     (меньше!)
  Y: 0.05     (меньше!)
  Z: 0.2      (короче!)
```

**Результат:** Оружие будет маленьким, в правом нижнем углу экрана ✅

## ✅ ИСПРАВЛЕНИЕ 2: УБРАТЬ MOBILE UI НА PC (30 секунд)

### Способ 1: В сцене Game

```
1. Unity → Scenes → Game
2. Hierarchy → найдите "MobileInputManager" или "MobileUI"
3. Inspector:
   - Если видите MobileInputManager:
     Auto Detect Platform: ✅ (должна быть включена!)
   
   - Если видите Canvas с мобильными кнопками:
     Просто отключите GameObject (снимите галочку слева от имени)
```

---

### Способ 2: Удалить Mobile UI из сцены

```
1. Unity → Scenes → Game
2. Hierarchy → найдите объекты:
   - MobileUI
   - VirtualJoystick
   - Mobile Buttons
3. Удалите их (Delete)
```

**Почему появляется на PC?**  
Скорее всего в сцене Game есть Canvas с мобильными элементами, который не настроен на автоматическое отключение.

## 🐛 TROUBLESHOOTING:

### Проблема: Всё ещё вижу только оружие
```
Решение:
1. Weapon → Position Z: сделайте ещё меньше (0.3)
2. WeaponModel → Scale: уменьшите ещё (0.03, 0.03, 0.15)
3. Camera → FOV: увеличьте до 90
```

### Проблема: Оружие слишком маленькое
```
Решение:
WeaponModel → Scale: увеличьте (0.08, 0.08, 0.25)
```

### Проблема: Mobile UI всё ещё показывается
```
Решение:
1. Hierarchy → найдите все Canvas объекты
2. Проверьте какой из них содержит джойстик
3. Отключите этот Canvas (снимите галочку в Inspector)
```
