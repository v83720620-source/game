## 🎯 КАК РАБОТАЕТ:

### **1. NO WALLHACKS (Last Known Position)**

```
Scenario:
1. Bot видит игрока → Запоминает позицию
2. Игрок прячется за стеной
3. Bot теряет Line of Sight
4. Bot стреляет в Last Known Position (где видел последний раз)
5. Через 3s → Bot забывает позицию
6. Bot перестаёт стрелять

Без Fair Play (чит):
✅ Bot всегда знает позицию → Wallhack

С Fair Play:
✅ Bot помнит 3s → Реалистично
✅ Потом забывает → Честно
```

### **2. SMOKE RESPECT**

```
Scenario:
1. Игрок в дыму
2. Bot проверяет: IsTargetInSmoke()
3. Если в дыму:
   - Accuracy * 0.3 (снижена на 70%)
   - 30% chance to shoot
   - Стреляет редко и неточно

Без Fair Play (чит):
✅ Bot стреляет точно сквозь дым

С Fair Play:
✅ Bot не видит в дыму
✅ Стреляет наугад
```

### **3. 360 NO-SCOPE BAN**

```
Scenario:
1. Игрок сзади бота (180° угол)
2. Bot пытается развернуться
3. Smooth rotation (Slerp) вместо instant
4. Минимум 0.2s на разворот на 180°
5. CanInstantShoot() = false
6. Bot не стреляет до полного разворота

Без Fair Play (чит):
✅ Bot мгновенно разворачивается → 360 no-scope

С Fair Play:
✅ Реалистичное время разворота
✅ Не может instant shoot за спиной
```

### **4. MERCY RULE**

```
Scenario:
1. Player умер от бота 3 раза подряд
2. MercyRuleSystem.RegisterPlayerDeath()
3. Проверка: 3 deaths в течение 60s?
4. Если ДА:
   - Mercy Rule triggered!
   - Bot difficulty: Hard → Normal
   - Accuracy снижена
   - Reaction time увеличено
5. Player получает kill:
   - Mercy Rule reset
   - Bot difficulty восстановлена

Результат:
✅ Игрок не фрустрируется
✅ Адаптивная сложность
✅ Незаметно для игрока (subtle)
```

## 🔧 ИНТЕГРАЦИЯ С BOTAI:

### **В BotAI.cs нужно добавить:**

```csharp
// References
private BotFairPlay _fairPlay;
private LastKnownPosition _lastKnownPos;

void Awake()
{
    _fairPlay = GetComponent<BotFairPlay>();
    _lastKnownPos = GetComponent<LastKnownPosition>();
}

void Update()
{
    if (_currentTarget != null)
    {
        // Update fair play tracking
        _fairPlay?.UpdateTargetTracking(_currentTarget);
        _lastKnownPos?.SetTarget(_currentTarget);
        
        // Get aim position (Last Known Position)
        Vector3 aimPos = _fairPlay?.GetTargetPosition() ?? _currentTarget.position;
        
        // Check if should shoot
        bool canShoot = _fairPlay?.ShouldShoot(aimPos) ?? true;
        
        if (canShoot)
        {
            // Shoot logic
            TryShoot();
        }
    }
}
```
