## 🎯 ОСОБЕННОСТИ СИСТЕМЫ:

### **1. Queue System**
```
✨ Auto matchmaking
✨ Timeout 15s → Add bots
✨ Timeout 60s → Force start
✨ Multiple game modes support
✨ Server-authoritative
```

### **2. Поддерживаемые режимы:**
```
- Duel 1v1      (2 players)
- TDM 3v3       (6 players)
- TDM 5v5       (10 players - future)
- Hardpoint 5v5 (10 players - future)
- Practice      (1 player + bots)
```

### **3. Логика работы:**
```
1. Player выбирает режим
2. Нажимает "Find Match"
3. Система ищет игроков (15s)
4. Если недостаточно → добавляет ботов
5. Если 60s прошло → запускает матч
6. Загружает GameScene
7. Спавнит игроков + ботов
```

## 📋 НАСТРОЙКА В UNITY:

### **ШАГ 1: Создать Lobby Scene**

1. **File → New Scene**
2. **Сохраните как:** `Assets/_Project/Scenes/LobbyScene.unity`

---

### **ШАГ 2: Настроить MatchmakingManager**

1. **Создайте пустой GameObject в сцене:**
   ```
   Hierarchy → Create Empty → "MatchmakingManager"
   ```

2. **Добавьте компоненты:**
   ```
   - NetworkObject (Unity Netcode)
   - MatchmakingManager (наш скрипт)
   ```

3. **Настройте NetworkObject:**
   ```
   ✓ Don't Destroy With Owner = true
   ✓ Destroy With Scene = false
   ```

4. **Настройте MatchmakingManager:**
   ```
   ✓ Auto Fill Bots = true
   ```

## 📊 ОЖИДАЕМОЕ ПОВЕДЕНИЕ:

### **Scenario 1: Instant Match (Full Lobby)**
```
Players join: 2/2 (Duel) or 6/6 (TDM)
→ Match starts instantly
→ No bots added
```

### **Scenario 2: Bot Fill (Timeout 15s)**
```
Players join: 1/2
Wait 15s
→ "Adding bots..."
→ 1 bot spawned
→ Match starts
```

### **Scenario 3: Force Start (Timeout 60s)**
```
Players join: 1/6 (TDM)
Wait 60s
→ "Match starting!"
→ 5 bots spawned
→ Match starts (even not full)
```

### **Scenario 4: Cancel Queue**
```
Player clicks [Cancel]
→ Removed from queue
→ Queue Panel hides
→ Can select mode again
```

## 🔧 TROUBLESHOOTING:

### **Проблема: "MatchmakingManager not found!"**
```
Решение:
1. Убедитесь что MatchmakingManager в сцене
2. Проверьте что NetworkObject добавлен
3. Проверьте что Singleton создаётся (Awake)
```

### **Проблема: "Queue Panel не появляется"**
```
Решение:
1. Проверьте ссылки в MatchmakingUI Inspector
2. Убедитесь что Queue Panel изначально SetActive = false
3. Проверьте Console на ошибки
```

### **Проблема: "Match не запускается"**
```
Решение:
1. Проверьте что GameScene в Build Settings
2. Проверьте Network Scene Manager настроен
3. Проверьте Console на ошибки
```
